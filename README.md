# CS67lib

**CS67lib** is a C library inspired by [CS50](https://cs50.harvard.edu/) and designed for students at RUPP to **practice memory management and low-level programming concepts**. It provides hands-on tools for working with dynamic arrays, strings, and linked lists, helping users understand safe and efficient memory use.

## Features
- **Dynamic Arrays:** array_init, array_push, array_get memory safely  
- **String Utilities:** cloning, trimming, joining, and safe manipulation  
- **Linked Lists:** add, remove, traverse, and manage memory manually
