# STLlibc
**MINI_STLlibc** is a C library inspired by me and designed for students like me at RUPP to **practice memory management and low-level programming concepts**. It provides hands-on tools for working with dynamic arrays, strings, and linked lists, helping me to understand memory management. I'm also planning to add more features like important DSA in computer science, and also add human written tests because tests and builds are being generated by AI :(

## Features Overview
```c
/* Dynamic array */
Vector* vector_init(size_t element, size_t capacity);
int vector_push(Vector *a, const void* element);
void* vector_get(Vector* vec, size_t index);
void vector_free(Vector *vec);
/* String */
int str_equal(const char *a, const char *b);
int str_len(const char* s);
char *str_copy(char* dest, const char* src);
char *str_chr(const char *str, const char uc);
char* str_cat(char* dest, const char* src);
char *str_clone(const char *s);
```

## Vector

A generic dynamic array inspired by `std::vector` from C++. Works with **any type** using `void*` — just like `malloc` hands you a raw address and trusts you to know what's there.

### Initialize
```c
Vector* v = vector_init(sizeof(int), 4);
```
| Parameter | Description |
|-----------|-------------|
| `element` | Size of each element in bytes — use `sizeof` |
| `capacity` | Initial capacity of the vector |
| returns | Pointer to the new vector, or `NULL` on failure |

### Push
```c
int x = 42;
vector_push(v, &x);
```
Appends an element to the end. Automatically doubles capacity when full.

| Parameter | Description |
|-----------|-------------|
| `vec` | Pointer to the vector |
| `element` | Pointer to the element to push |
| returns | `0` on success, `-1` on failure |

### Get
```c
int* val = (int*)vector_get(v, 0);
printf("%d\n", *val); // 42
```
Returns a `void*` to the element at the given index — cast it to your type.

| Parameter | Description |
|-----------|-------------|
| `vec` | Pointer to the vector |
| `index` | Zero-based index |
| returns | Pointer to element, or `NULL` if out of bounds |

### Free
```c
vector_free(v);
```
Frees all memory associated with the vector. Always call this when done.

### Full Example
```c
#include "vector.h"

typedef struct {
    char name[50];
    int grade;
} Student;

int main() {
    Vector* students = vector_init(sizeof(Student), 4);

    Student s1 = {"Alice", 90};
    Student s2 = {"Bob", 85};
    Student s3 = {"Charlie", 92};

    vector_push(students, &s1);
    vector_push(students, &s2);
    vector_push(students, &s3);

    for (size_t i = 0; i < students->size; i++) {
        Student* s = vector_get(students, i);
        printf("Name: %s, Grade: %d\n", s->name, s->grade);
    }

    vector_free(students);
    return 0;
}
```
